## 📌 Project Overview

Student dropout is a significant challenge faced by educational institutions. This project uses Machine Learning techniques to predict whether a student is at risk of dropping out based on academic, demographic, and socioeconomic factors. Early prediction enables institutions to provide timely support, improving student retention and academic success.

The project follows a complete machine learning workflow, including data preprocessing, exploratory data analysis (EDA), feature engineering, model training, evaluation, and prediction.
# 📂 Dataset

The project uses the **EduSpark Student Dropout Dataset**.

### Features

- Academic Score
- Attendance Percentage
- Course Engagement
- Family Income
- Gender
- Region
- Parental Education
- Dropout Status (Target Variable)
